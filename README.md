# MyCloudMusic
仿网易云音乐WPF开发

用的是某大神的网易云音乐的API：https://binaryify.github.io/NeteaseCloudMusicApi/#/

部署在搭档的服务器内（已注销）
使用WPF写了一些简单的功能，能够用真实的网易云账号登录
